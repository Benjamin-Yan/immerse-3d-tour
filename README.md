### Name
`Immerse 3D tour`

### Features
- `移動搖桿`: 方便手機使用者在場景中移動。
- `相機視角切換`: 給定一組特定的觀賞視角，方便使用者欣賞。
- `場景介紹`: 簡單介紹場景，並附上操作說明。
- `場景選擇`: 使用者可以簡單地跳轉到他們想要觀看的場景，或是回到首頁。 (目前共5個場景)

### Reference
- `gsplat`: [github](https://github.com/huggingface/gsplat.js)
